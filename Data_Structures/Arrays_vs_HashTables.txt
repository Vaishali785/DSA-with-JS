* Arrays
    1. Lookup - O(1)
    2. Push - O(1)
    3. Insertion - O(n)
    4. Deletion - O(n)
    5. Search - O(n)

* HashTables 
    1. Lookup - O(1)
    2. Insertion - O(1)
    3. Deletion - O(1)
    4. Search - O(1)



1. Array - ordered, HashTables - unordered
2. Array - index(we can't customize), HashTables - keys (we can customize)
