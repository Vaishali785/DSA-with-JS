~ Linked list is a list which is linked. Each node of linked list contains two values - value, pointer 
~ Pointer of each item points to the value of next item. 
~ Starting point is called head and last element is called tail.

apples

                133  --------> pears 
                                12 -------> null 

~ In JS, there is no pre-built linked list.
